# SurrealDB (surrealdb)

SurrealDB is a multi-model database that unifies documents, graphs, vectors, time-series, full-text search, and relational data within a single ACID transaction framework. It exposes a native HTTP REST API and supports SurrealQL, a unified query language for multi-model data access. SurrealDB Cloud provides managed hosting with usage-based pricing, while the open-source engine supports self-hosted deployments. The platform also includes a Model Context Protocol (MCP) server enabling AI tools and agents to interact with the database directly.
